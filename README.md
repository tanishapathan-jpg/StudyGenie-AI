# StudyGenie- AI

## AI-Powered Personal Study Mentor

StudyGenie AI is an intelligent educational assistant built using IBM watsonx Orchestrate and IBM Granite Models.

## Features

- AI Tutor
- Retrieval-Augmented Generation (RAG)
- PDF Question Answering
- Quiz Generator
- Flashcard Generator
- Study Planner
- Notes Summarization
- Personalized Learning

## Technology Stack

- IBM Cloud
- IBM watsonx Orchestrate
- IBM Granite
- Retrieval-Augmented Generation (RAG)
- Knowledge Base
